O mně

Mgr. Luděk Kostera

Web kodér, Programator

Profil

Jmenuji se Luděk Kostera. Zabývám se programováním a tvorbou webových stránek.

Během let jsem se perfektně naučil HTML, CSS a Javascript. Pro efektivnější psaní CSS používám pre-processory LESS a SASS. Pracuji s jQuery, Angular frameworkem a mé šablony jsou vytvářeny s využitím Bootstrapu.

Aplikace programuji v jazycích JAVA, PHP. Programoval jsem i v .NET (C#) a Flash (Action Script).

Zaměření

Tvorba webu

Specializuji se především moderní a čistý design.

Kóduji validně, s ohledem na SEO. Dbám na použitelnost, přístupnost a sémantiku kódu.

Při návrhu a realizaci dodržuji zásady, které jsou základním předpokladem pro kvalitní webové stránky. Jsou to nalezitelnost, dostupnost, přístupnost a použitelnost.

Programovaní

Znám principy objektově orientovaného programování a návrhové vzory.

Při programování aplikaci upřednostňuji jazyk Java a ovládám na pokročilé úrovni PHP,

Mám zkušenosti s databází Oracle, ovládám dotazovací jazyk SQL na úrovní návrhu tabulek a práci s nimi, psaní pokročilých dotazů.
