O mně

Mgr. Luděk Kostera

Web Designer, Programator

Profil

Jmenuji se Luděk Kostera. Zabývám se programováním a tvorbou webových stránek.

Během let jsem se jako kodér perfektně naučil HTML, CSS a Javascript. Pro efektivnější psaní CSS používám pre-processory LESS a SASS. Pracuji s jQuery, Angular frameworkem a mé šablony jsou vytvářeny s využitím Bootstrapu.

Aplikace programuji v jazycích JAVA, PHP. Programoval jsem i v .NET (C#) a Flash (Action Script).

Zaměření

Tvorba webu

Specializuji se především moderní a čistý design.

Kóduji validně, s ohledem na SEO. Dbám na použitelnost, přístupnost a sémantiku kódu.

Tvorba webu na míru zahrnuje celý proces od prvotního zadání po spuštěný web na internetu. Při návrhu a realizaci dodržuji zásady, které jsou základním předpokladem pro kvalitní webové stránky. Jsou to nalezitelnost, dostupnost, přístupnost a použitelnost.

Programovaní

Znám principy objektově orientovaného programování a návrhové vzory.

Při programování aplikaci upřednostňuji jazyk Java.

Co se ostatních platforem týče, ovládám na pokročilé úrovni jazyky PHP, C# a ActionScript(Flash).

Mám zkušenosti s databází Oracle, ovládám dotazovací jazyk SQL na úrovní návrhu tabulek a práci s nimi, psaní pokročilých dotazů.
