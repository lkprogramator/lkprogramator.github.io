HR solution

 

AngularJS, Bootstrap, CSS3, Javascript, Java

Jeden z projektů, na kterých jsem se podílel byl HR solution. HR(Human resource) řešení pro výběr vhodných kandidátů na pracovní pozici a jejich zapracování, či sestavení vhodného pracovního týmu.

Front-end je vytvořen Angularovskou aplikací, napojenou na REST api zprogramované v javě.

A s čím vším jsem na projektu pomáhal: Programování rest api v javě, Tvorba front-end aplikace s využitím angularJS frameworku, včetně implementace grafického vzhledu podle zadání.
