O mně

Mgr. Luděk Kostera

Web kodér, Programator

Zabývám se programováním a tvorbou webových stránek. Během let jsem se perfektně naučil HTML, CSS a Javascript. Pro efektivnější psaní CSS používám pre-processory LESS a SASS. Pracuji s Angular frameworkem a mé šablony jsou vytvářeny s využitím Bootstrapu. Aplikace programuji v jazycích JAVA, TypeScript. Programoval jsem i v .NET (C#) a Flash (Action Script).
